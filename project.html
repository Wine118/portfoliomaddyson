Restaurant Menu & Ordering Web Application

A full-stack restaurant menu and ordering system built using HTML, CSS, JavaScript, Django, and MySQL. The application supports dynamic menu management, customer order processing, and persistent database storage.

  • Developed a fully responsive frontend optimized for mobile and desktop devices.
  • Implemented backend logic using Django with structured MVT architecture and Django ORM.
  • Designed and maintained consistent relational data models using MySQL.
  • Built dynamic menu rendering with category and dish management.
  • Implemented order validation and secure data handling for reliable transaction processing.
